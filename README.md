# Эрэмбэлэх Алгоритмууд (Sorting Algorithms)

Энэхүү төсөл нь хамгийн түгээмэл ашиглагддаг хоёр үндсэн эрэмбэлэх алгоритмыг Python хэл дээр хэрэгжүүлэхэд зориулагдсан.

## 1. Эрэмбэлэх алгоритм гэж юу вэ?
Эрэмбэлэх алгоритм нь жагсаалт эсвэл массив дахь өгөгдлийг тодорхой дараалалд (ихэвчлэн өсөх эсвэл буурах) оруулах үйлдлийг хэлнэ. Энэ нь өгөгдөл хайх, мэдээллийг цэгцлэх зэрэгт компьютерын шинжлэх ухааны хамгийн суурь үйлдэл юм.

---

## 2. Bubble Sort (Хөөсөн эрэмбэлэлт)

**Bubble Sort** нь зэргэлдээ орших хоёр элементийг харьцуулж, хэрэв тэдгээр нь буруу дараалалтай байвал байрыг нь солих замаар ажилладаг энгийн алгоритм юм. Энэ үйлдэл нь жагсаалт бүрэн эрэмбэлэгдэх хүртэл давтагдана.



### Хэрэгжүүлэх алхам:
1. Массивын эхнээс эхлэн зэргэлдээх хоёр элементийг харьцуулна.
2. Хэрэв зүүн талын элемент баруун талынхаас их бол байрыг нь солино.
3. Энэ үйлдлийг массивын төгсгөл хүртэл хийнэ (ингэснээр хамгийн их утга хамгийн ард "хөөс" мэт гарч ирнэ).
4. Массив бүрэн эрэмбэлэгдэх хүртэл дээрх алхмуудыг давтана.

**Хугацааны нарийн төвөгтэй байдал:** O(n^2)

---

## 3. Selection Sort (Сонгох эрэмбэлэлт)

**Selection Sort** нь өгөгдсөн жагсаалтыг хоёр хэсэгт хувааж үздэг: эрэмбэлэгдсэн хэсэг болон эрэмбэлэгдээгүй хэсэг. Алгоритм нь эрэмбэлэгдээгүй хэсгээс хамгийн бага элементийг олж, түүнийг эрэмбэлэгдсэн хэсгийн төгсгөлд шилжүүлдэг.



### Хэрэгжүүлэх алхам:
1. Массивын эхний элементийг хамгийн бага (minimum) гэж үзнэ.
2. Үлдсэн элементүүд дундаас хамгийн бага утгатай элементийг хайна.
3. Олдсон хамгийн бага элементийг массивын эхний элементтэй байрыг нь солино.
4. Дараагийн элемент рүү шилжиж, массив дуустал энэ үйлдлийг давтана.

**Хугацааны нарийн төвөгтэй байдал:** O(n^2)
